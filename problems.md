< 문제 > 

[ SW0 ] (상승 Edge) 한번 누를 시 count 1씩 증가, count값 FND로 출력

[ SW1 ] (하강 Edge) 한번 누를 시 count 1씩 감소, count값 FND로 출력

[ SW2 ] (상승 Edge) 1번 : LED 왼쪽 shift  /  2번 : LED 오른쪽 shift

[ SW3 ] (하강 Edge) 누를 때 마다 왼쪽으로 Shift 되는 간격 증가(1~4칸 까지 시프트)
